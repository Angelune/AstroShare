# Astroshare :

Réseau social (sans utilisateurs) de partage d'astrophotographies

Installation :

1. Cloner le repo :

```sh
git clone git@github.com:Angelune/AstroShare.git
```

2. Installer les dépendances :

```sh
cd projet-2022-weinreich-rondeau-gr3
npm ci
```

3. Lancer le projet :

```sh
npm start
```
